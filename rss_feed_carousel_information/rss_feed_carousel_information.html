Overview

Written by: Larry Robertson on December 17, 2017

This document contains links to the information gathered by me to supplement any existing information or documentation provided by Tightrope Media Systems as related to the Tightrope Carousel version 6.4.5.

The Three Most Common Errors in Carousel When Creating RSS Feeds

  1. Cause: RSS Feed File not Found on Server. Carosel reports error as: The supplied syndication resource has a content format of None, which does not match the expected content format of Rss.
  2. Cause: One or more Blocks are overlaping one another. Carosel reports error as: Error: No blocks could fit on the current template.
  3. Cause: Either the channel image url is invalid or the item enclosure url attribute value is invalid. Carosel reports error as: Error: Please contact Administrator's Name for assistance.
